This sample visualizes the data about electricity production using stacked stepped area chart.

In this example, you can see how to render and configure the stacked stepped area chart. This series forms the step area progress, by connecting points through vertical and horizontal lines with area filled. You can use fill properties to customize the area. marker and dataLabel are used to represent individual data and its value.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use stacked stepped area series, we need to inject StackingStepAreaSeriesService into the @NgModule.providers section.

More information on the area types series can be found in this documentation section.
